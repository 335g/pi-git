## 場所
`src/core/git.ts:109-125`

## 問題
`git stash push` は成功（code 0）したが、`git stash show -p stash@{0}` が失敗（code ≠ 0）した場合、`stashDiff` は空文字列になり、そのまま `diff = ""` として処理が続行される。呼び出し元 (`agg-commit.ts:93`) では `diff.trim() === ""` → 「No changes to commit」と表示され、実際のエラーがマスクされる。

`git stash push` と同様に、`stash show` の終了コードもチェックすべき。

## 修正案
```typescript
const { stdout: stashDiff, code: showCode } = await pi.exec(
    "git", ["stash", "show", "-p", "stash@{0}"], { cwd }
);
if (showCode !== 0) {
    // handle error, likely return null
}
diff = stashDiff;
```
