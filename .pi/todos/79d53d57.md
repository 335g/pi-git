## 使用方法

```bash
# accumulate モードに切り替え
/git-config auto_agg_commit_mode accumulate

# 変更が蓄積される（各 turn 終了時に commit されない）
# footer に "auto-commit: accumulate (3ターン) | 5ファイル" と表示

# まとめてコミット
/git-agg-commit

# レビュー付きでコミット
/git-agg-commit --review
```
