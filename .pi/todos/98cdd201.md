## 場所
`src/core/diff-analyzer.ts:300`

## 問題
```ts
for (let i = 0; i < batches.length; i++) {
    void footerManager.setCommitProgress(i + 1, batches.length);  // ← 浮遊 Promise
    try {
        const hunks = await callAIForDiff(...);
```

`footerManager.setCommitProgress()` は `async` 宣言されている（`Promise<void>` を返す）。`void` 演算子で Promise を捨てているため、もし内部で `renderPhase()` → `setStatus()` が例外を投げると**未処理の Promise Rejection** になり Node.js が `UnhandledPromiseRejectionWarning` を出す。

## 修正案
- `setCommitProgress` を `await` する（内部は同期的なのでパフォーマンス影響なし）
- または `setCommitProgress` から不要な `async` を外して同期メソッドにする（`footerManager` の他のメソッドも同様）
